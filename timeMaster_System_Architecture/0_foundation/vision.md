# 项目愿景（Vision）

我们正在构建一个连接“需求”和“能力”的本地服务平台。

现实世界中存在两个长期未被很好解决的问题：

1. 有需求的人，无法快速找到合适的人来完成任务
2. 有能力的人，无法稳定获得匹配的任务与收入

本项目的目标是：

通过“技能标签 + 时间调度 + 地理位置”的核心机制，
建立一个高效、可信、可持续的服务撮合系统，使：

* 用户可以快速发布需求并获得响应
* 服务者可以基于自身技能稳定接单
* 平台可以持续优化匹配效率

长期愿景：

随着系统的发展，我们将逐步沉淀用户的能力数据，
构建更精准的匹配系统与个性化推荐能力，
提升整个社会的服务效率。

# 当前版本定义（Version）

版本：v0.1 MVP

当前阶段目标：

* 完成基础任务撮合闭环
* 支持实时任务匹配
* 支持简单优先级（普通 / 紧急）

当前不包含：

* 数字人系统
* 复杂AI能力
* 硬件数据接入

核心能力：

* 技能标签匹配
* 时间调度
* 地理位置匹配
