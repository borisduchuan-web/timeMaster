# 产品定义（Product Definition）

一句话定义：

这是一个基于技能标签与时间调度的本地即时服务撮合平台。

核心功能：

* 用户发布任务（时间 + 地点 + 内容 + 价格）
* 系统匹配合适的服务者
* 服务者接单并完成任务
* 平台提供评价与保障机制

核心价值：

* 提高需求与服务的匹配效率
* 降低用户寻找服务的成本
* 提升服务者的接单稳定性

与传统平台的区别：

* 更强调“时间匹配”而不是仅仅分类信息
* 更强调“技能标签”而不是简单身份
* 逐步构建数据驱动的匹配能力（而非纯人工选择）

当前阶段（MVP）不包含：

* 复杂AI建模
* 全量用户数据采集
* 硬件/传感器接入

这些能力将在后续阶段逐步引入。
