# timeMaster Prototype

This folder contains clickable static prototypes for the first product review.

## Files

```text
app.html       Mobile app prototype
admin.html     PC admin console prototype
prototype.css  Shared styles
```

## Open Locally

Open the files directly in a browser:

```text
docs/prototype/app.html
docs/prototype/admin.html
```

No build step or server is required.
